Detailed Description

template<typename T, bool DeserializeChunk = true>
class vanetza::asn1::PacketVisitor< T, DeserializeChunk >

PacketVisitor is a helper to extract the ASN.1 wrapper object from a packet's application layer. This is a generalised version of the implementation found in Artery before.

ChunkPacket's application layer is simply casted for the sake of speed whenever possible. When casting fails, this helper can fall back to deserialization, see DeserializationChunk parameter. Deserialization is the only option for CohesivePacket's application layer.
